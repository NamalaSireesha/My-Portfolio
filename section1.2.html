HEALTHGUARD1.0

Healthguard1.0 is the next genertion health care system which consists of 9 different module. Each module is different that depicts different meaning.

    1.Personalized Health Assessment

  • This module stores the details of the previous health assessments of the patients and is used to create new assessment when the user takes treatement.

    • 2.Predictive Analysis Navigation

  • This module is a visual representation of the patient health details like heartrate , BP etc.....
  • It is also used to share alerts and notifications to the patient.

    • 3.Early Warning System

  • It consists of Real-time monitoring vital signs and represent the patient details in graphical way

    • 4.Behavioral Modification Programs

  • This module consists of registration forms for certain programmes like Weight Loss , Smoking Cessation , Stress Management etc....
  • It will help the user to control his/her health by participating in those programmes

    • 5.Collaborative Care Teams

  • It consists of the detils of the doctor so that the user can contact if there is any need

    • 6.Data Privacy and Security

  • The last module is used for data Privacy and Security
